# Project Euler Solutions

Personal solutions for [Project Euler](https://projecteuler.net/) problems.

## Layout

```text
.
+-- solutions/
    +-- problem_001/
    |   +-- problem_001.py
    +-- problem_080/
    |   +-- problem_080.py
    +-- problem_085/
    |   +-- problem_085.py
    +-- problem_095/
    |   +-- problem_095.py
    +-- problem_096/
    |   +-- problem_096.py
    |   +-- problem_096_sudoku.txt
    +-- problem_099/
    |   +-- problem_099.py
    |   +-- problem_099_base_exp.txt
    +-- problem_206/
        +-- problem_206.py
```

Each file is a standalone script. Keep the code simple and easy to run.

## Run a Solution

```powershell
python solutions/problem_XXX/problem_XXX.py
```
