# Project Euler Solutions

Personal solutions for [Project Euler](https://projecteuler.net/) problems.

## Layout

```text
.
+-- solutions/
    +-- problem_XXX/
        +-- problem_XXX.py
        +-- problem_XXX_input.txt
```

Each problem has its own folder. The solution file is a standalone script, and
any input files live beside it.

## Run a Solution

```powershell
python solutions/problem_XXX/problem_XXX.py
```
